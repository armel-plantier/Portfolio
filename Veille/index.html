Vulnérabilités et Correctifs

  • Alerte CISA sur le malware RESURGE exploitant Ivanti Connect Secure : L'agence américaine CISA a publié de nouveaux détails concernant RESURGE, un implant malveillant utilisé dans des attaques zero-day exploitant la vulnérabilité CVE-2025-0282 pour compromettre les dispositifs Ivanti Connect Secure. (https://www.bleepingcomputer.com/news/security/cisa-warns-that-resurge-malware-can-be-dormant-on-ivanti-devices/)
  • Vulnérabilités critiques Trend Micro Apex One : Trend Micro a corrigé deux vulnérabilités critiques dans Apex One permettant à des attaquants d'exécuter du code à distance (RCE) sur des systèmes Windows vulnérables. (https://www.bleepingcomputer.com/news/security/trend-micro-warns-of-critical-apex-one-rce-vulnerabilities/)
  • Vulnérabilité critique Juniper Networks PTX : Une vulnérabilité critique dans le système d'exploitation Junos OS Evolved des routeurs PTX Series de Juniper Networks pourrait permettre à un attaquant non authentifié d'exécuter du code à distance avec les privilèges root. (https://www.bleepingcomputer.com/news/security/critical-juniper-networks-ptx-flaw-allows-full-router-takeover/)
  • Clés API Google exposant des données Gemini AI : Des clés API Google, auparavant considérées comme inoffensives et intégrées dans du code client-side accessible, pourraient désormais être utilisées pour s'authentifier à l'assistant Gemini AI et accéder à des données privées. (https://www.bleepingcomputer.com/news/security/previously-harmless-google-api-keys-now-expose-gemini-ai-data/)
  • Améliorations de sécurité pour les fichiers batch Windows 11 : Microsoft déploie de nouvelles versions de test pour Windows 11 Insider Preview qui améliorent la sécurité et les performances lors de l'exécution de fichiers batch ou de scripts CMD. (https://www.bleepingcomputer.com/news/microsoft/microsoft-testing-windows-11-batch-file-security-improvements/)

(https://www.bleepingcomputer.com/news/security/quicklens-chrome-extension-steals-crypto-shows-clickfix-attack/)

(https://www.bleepingcomputer.com/news/security/48m-in-crypto-stolen-after-korean-tax-agency-exposes-wallet-seed/)

(https://www.bleepingcomputer.com/news/security/european-dyi-chain-manomano-data-breach-impacts-38-million-customers/)
  • L'Olympique de Marseille confirme une "tentative" de cyberattaque : Le club de football professionnel français Olympique de Marseille a confirmé une cyberattaque après qu'un acteur de la menace ait affirmé avoir violé les systèmes du club plus tôt ce mois-ci. (https://www.bleepingcomputer.com/news/security/olympiqueparseille-football-club-confirms-cyberattack-after-data-leak/)

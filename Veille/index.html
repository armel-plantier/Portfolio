🛡️ Journal de Veille : du 23/02/2026 au 01/03/2026

Dimanche 1 Mars

Données Personnelles / Vie Privée

Samsung a conclu un accord avec l'État du Texas pour cesser la collecte d'informations sur le visionnage de contenu via ses Smart TV sans consentement explicite. 🔗 Source

Samedi 28 Février

Attaques / Extensions de navigateur

L'extension Chrome "QuickLens - Search Screen with Google Lens" a été retirée du Chrome Web Store après avoir été compromise. Elle distribuait des malwares et tentait de voler des cryptomonnaies à des milliers d'utilisateurs. 🔗 Source

Incidents de Sécurité / Fuite de données

Le Service National des Impôts de Corée du Sud a accidentellement exposé la phrase de récupération mnémonique d'un portefeuille de cryptomonnaies saisi dans un communiqué de presse officiel, permettant à des attaquants de voler l'équivalent de 4,8 millions de dollars en cryptomonnaie. 🔗 Source

Vendredi 27 Février

Mises à jour / Sécurité OS

Microsoft déploie de nouvelles versions de Windows 11 Insider Preview qui améliorent la sécurité et les performances lors de l'exécution de fichiers batch ou de scripts CMD. 🔗 Source

Menaces Persistantes Avancées (APT) / Malware

Des hackers nord-coréens du groupe APT37 déploient de nouveaux outils malveillants pour exfiltrer des données entre des systèmes connectés à Internet et des réseaux isolés (air-gapped). Ces malwares se propagent via des lecteurs amovibles et mènent des activités de surveillance discrète. 🔗 Source

Cybercriminalité / Opérations policières

Une opération coordonnée par Europol, baptisée "Project Compass", a mené à 30 arrestations et identifié 179 suspects liés à "The Com", un collectif de cybercriminalité ciblant les enfants et adolescents. 🔗 Source

Malware / Vulnérabilités

La CISA (U.S. Cybersecurity and Infrastructure Security Agency) a publié de nouveaux détails sur RESURGE, un implant malveillant utilisé dans des attaques zero-day exploitant la vulnérabilité CVE-2025-0282 pour compromettre les dispositifs Ivanti Connect Secure. Le malware peut rester dormant sur les appareils Ivanti. 🔗 Source

Gestion des Vulnérabilités / Bonnes Pratiques

Un article met en lumière l'importance du patching des logiciels tiers (lecteurs PDF, clients de messagerie, utilitaires d'archivage) qui définissent la surface d'attaque réelle. Une gestion cohérente de ces mises à jour est essentielle pour réduire les risques d'exploitation sur les endpoints. 🔗 Source

Cybercriminalité / Fraude

Un Ukrainien a plaidé coupable d'avoir géré OnlyFake, un site web qui utilisait l'intelligence artificielle pour générer et vendre plus de 10 000 photos de fausses pièces d'identité à des clients du monde entier. 🔗 Source

Jeudi 26 Février

Vulnérabilités / Sécurité API

Des clés API Google pour des services comme Maps, précédemment considérées comme inoffensives lorsqu'elles sont intégrées dans du code côté client, pourraient désormais être utilisées pour s'authentifier auprès de l'assistant Gemini AI et accéder à des données privées. 🔗 Source

Vulnérabilités / Patchs de Sécurité

Trend Micro a corrigé deux vulnérabilités critiques dans son produit Apex One qui permettaient l'exécution de code à distance (RCE) sur les systèmes Windows vulnérables. 🔗 Source

Incidents de Sécurité / Fuite de données

La chaîne européenne de bricolage ManoMano a notifié ses clients d'une fuite de données personnelles impactant 38 millions d'entre eux, suite à la compromission d'un fournisseur de services tiers. 🔗 Source

Vulnérabilités / Réseaux

Une vulnérabilité critique a été découverte dans le système d'exploitation réseau Junos OS Evolved des routeurs PTX Series de Juniper Networks. Cette faille pourrait permettre à un attaquant non authentifié d'exécuter du code à distance avec les privilèges root et de prendre le contrôle total du routeur. 🔗 Source

Incidents de Sécurité / Cyberattaques

Le club de football professionnel français Olympique de Marseille a confirmé une "tentative" de cyberattaque après qu'un acteur malveillant a affirmé avoir pénétré les systèmes du club plus tôt ce mois-ci, entraînant une fuite de données. 🔗 Source

Tendances / Ransomware

Le taux de paiement des rançons a chuté à un niveau record de 28% l'année dernière, et ce malgré une augmentation significative du nombre d'attaques revendiquées par ransomware. 🔗 Source

Vulnérabilités et Alertes

  • Vulnérabilités critiques dans Trend Micro Apex One Trend Micro a corrigé deux vulnérabilités critiques permettant l'exécution de code à distance (RCE) sur les systèmes Windows via son produit Apex One. https://www.bleepingcomputer.com/news/security/trend-micro-warns-of-critical-apex-one-rce-vulnerabilities/

  • Faille critique dans Juniper Networks PTX Series Une vulnérabilité critique a été identifiée dans le système d'exploitation Junos OS Evolved des routeurs PTX Series de Juniper Networks. Cette faille permet à un attaquant non authentifié d'exécuter du code à distance avec les privilèges root, conduisant à une prise de contrôle complète du routeur. https://www.bleepingcomputer.com/news/security/critical-juniper-networks-ptx-flaw-allows-full-router-takeover/

  • Alerte CISA sur le malware RESURGE dormant sur les appareils Ivanti La CISA (U.S. Cybersecurity and Infrastructure Security Agency) a publié de nouveaux détails concernant RESURGE, un implant malveillant utilisé dans des attaques zero-day exploitant la vulnérabilité CVE-2025-0282 pour compromettre les appareils Ivanti Connect Secure. Il est signalé que ce malware peut rester dormant sur les systèmes infectés. https://www.bleepingcomputer.com/news/security/cisa-warns-that-resurge-malware-can-be-dormant-on-ivanti-devices/

  • Clés API Google exposant des données Gemini AI Des clés API Google, auparavant considérées comme inoffensives pour des services comme Maps et intégrées dans le code client-side, pourraient désormais être utilisées pour s'authentifier auprès de l'assistant Gemini AI et accéder à des données privées. https://www.bleepingcomputer.com/news/security/previously-harmless-google-api-keys-now-expose-gemini-ai-data/

  • Améliorations de sécurité pour les scripts batch de Windows 11 Microsoft déploie de nouvelles versions préliminaires de Windows 11 qui améliorent la sécurité et les performances lors de l'exécution de fichiers batch ou de scripts CMD. https://www.bleepingcomputer.com/news/microsoft/microsoft-testing-windows-11-batch-file-security-improvements/

Incidents et Attaques Majeures

  • Extension Chrome compromise pour le vol de cryptomonnaies L'extension Chrome "QuickLens - Search Screen with Google Lens" a été retirée du Chrome Web Store après avoir été compromise pour diffuser des malwares et tenter de voler des cryptomonnaies à des milliers d'utilisateurs. https://www.bleepingcomputer.com/news/security/quicklens-chrome-extension-steals-crypto-shows-clickfix-attack/

  • Vol de 4,8 millions de dollars en cryptomonnaies après l'exposition d'une clé de récupération par l'agence fiscale coréenne Le Service National des Impôts de Corée du Sud a accidentellement exposé la phrase de récupération mnémonique d'un portefeuille de cryptomonnaies saisi dans un communiqué de presse officiel, permettant à des hackers de voler 6,4 milliards de wons (4,8 millions de dollars) en cryptomonnaies. https://www.bleepingcomputer.com/news/security/48m-in-crypto-stolen-after-korean-tax-agency-exposes-wallet-seed/

  • Fuite de données chez ManoMano impactant 38 millions de clients La chaîne européenne de bricolage ManoMano a notifié ses clients d'une fuite de données personnelles, causée par la compromission d'un fournisseur de services tiers, affectant 38 millions de comptes. https://www.bleepingcomputer.com/news/security/european-dyi-chain-manomano-data-breach-impacts-38-million-customers/

  • L'Olympique de Marseille confirme une cyberattaque après une fuite de données Le club de football professionnel français Olympique de Marseille a confirmé une cyberattaque après qu'un acteur de la menace ait revendiqué avoir accédé aux systèmes du club plus tôt ce mois-ci, entraînant une fuite de données. https://www.bleepingcomputer.com/news/security/olympique-marseille-football-club-confirms-cyberattack-after-data-leak/

  • Le groupe APT37 utilise de nouveaux malwares pour cibler les réseaux isolés Les hackers nord-coréens du groupe APT37 déploient de nouveaux outils pour déplacer des données entre les systèmes connectés à Internet et les réseaux isolés, en se propageant via des lecteurs amovibles et en menant une surveillance secrète. https://www.bleepingcomputer.com/news/security/apt37-hackers-use-new-malware-to-breach-air-gapped-networks/

Vulnérabilités et Correctifs

  • CISA alerte sur le malware RESURGE latent sur les appareils Ivanti L'agence américaine CISA a publié de nouveaux détails concernant RESURGE, un implant malveillant utilisé dans des attaques zero-day exploitant la vulnérabilité CVE-2025-0282 pour compromettre les appareils Ivanti Connect Secure. Il est recommandé de vérifier la présence de ce malware dormant.

  • Trend Micro alerte sur des failles critiques d'exécution de code dans Apex One Trend Micro a corrigé deux vulnérabilités critiques dans Apex One permettant à des attaquants d'exécuter du code à distance (RCE) sur les systèmes Windows vulnérables.

  • Faille critique dans Juniper Networks PTX permet une prise de contrôle totale du routeur Une vulnérabilité critique dans le système d'exploitation réseau Junos OS Evolved des routeurs PTX Series de Juniper Networks pourrait permettre à un attaquant non authentifié d'exécuter du code à distance avec les privilèges root.

  • Des clés API Google auparavant inoffensives exposent désormais les données Gemini AI Des clés API Google pour des services comme Maps, intégrées dans du code côté client accessible, pourraient être utilisées pour s'authentifier auprès de l'assistant Gemini AI et accéder à des données privées.

Menaces et Attaques

  • Les hackers APT37 utilisent un nouveau malware pour violer les réseaux air-gapped Des hackers nord-coréens déploient de nouveaux outils découverts pour transférer des données entre des systèmes connectés à Internet et des systèmes air-gapped, se propageant via des lecteurs amovibles et menant une surveillance discrète.

  • Un Ukrainien plaide coupable pour avoir géré un site de fausses pièces d'identité alimenté par l'IA Un Ukrainien a plaidé coupable d'avoir exploité OnlyFake, un site web alimenté par l'IA qui générait et vendait plus de 10 000 photos de faux documents d'identité à des clients du monde entier.

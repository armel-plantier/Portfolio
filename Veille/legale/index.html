Lunettes connectées : la CNIL alerte sur les risques pour la vie privée

Résumé : La CNIL souligne les dangers des lunettes connectées, qui combinent collecte massive de données biométriques et surveillance intrusive. La CNIL lance un plan d’action pour encadrer leur usage et sensibiliser les utilisateurs.

Multiples vulnérabilités critiques dans les produits SAP

Résumé : Le CERT-FR identifie plusieurs failles critiques dans les solutions SAP, permettant des attaques par exécution de code arbitraire, déni de service et fuite de données. Ces vulnérabilités, non corrigées dans certaines versions, exposent les entreprises à des risques majeurs de cybersécurité. Une mise à jour urgente est recommandée.

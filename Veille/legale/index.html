### Vulnérabilité critique "Copy Fail" dans les distributions Linux (CVE-2026-3602)

Résumé : Une faille critique nommée "Copy Fail" (CVE-2026-3602) affecte toutes les distributions Linux majeures depuis 2017, permettant une exécution de code arbitraire via des opérations de copie de fichiers. Exploitée activement, elle a été ajoutée à la liste KEV de la CISA. Microsoft a observé des tentatives limitées d'exploitation, principalement pour des tests de preuve de concept.

### Acquisition de Astrix Security par Cisco pour renforcer la sécurité des identités non humaines

Résumé : Cisco annonce l'acquisition d'Astrix Security pour 1,2 milliard de dollars, afin de renforcer sa position dans la sécurité centrée sur l'identité pour les accès machine et l'IA. Cette opération s'inscrit dans une stratégie plus large visant à sécuriser les environnements hybrides et cloud, en réponse à l'augmentation des risques liés aux identités non humaines.

### Violation du dépôt de code source de Trellix : impact limité confirmé

Résumé : Trellix révèle qu'un attaquant a compromis son dépôt de code source, mais l'enquête interne n'a révélé aucun impact sur les versions distribuées ou le processus de publication. L'incident souligne les risques croissants de ciblage des dépôts de code, notamment dans le secteur de la cybersécurité.

### DigiCert révoque des certificats après une intrusion via son portail de support

Résumé : DigiCert a révoqué des certificats après qu'un attaquant ait compromis son portail de support client. Les pirates ont utilisé un canal de chat pour livrer un malware, infectant un système d'analyste et accédant à l'outil interne. Aucune preuve d'impact sur les certificats émis n'a été confirmée.

### Exploitation massive de CVE-2026-41940 : plus de 40 000 serveurs cPanel compromis

Résumé : Une campagne d'exploitation active cible une faille zero-day (CVE-2026-41940) dans cPanel, permettant aux attaquants d'obtenir un accès administratif. Plus de 40 000 serveurs ont été compromis, soulignant l'urgence de patcher cette vulnérabilité récemment corrigée.

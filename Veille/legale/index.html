Lundi 30 Mars 2026

Opération Cactus 2026 : renforcer durablement la sensibilisation de la communauté éducative aux cybermenaces

Résumé : Le ministère de l’Éducation nationale a mené une opération nationale de prévention par simulation d’hameçonnage pour sensibiliser la communauté éducative aux cybermenaces. Cette initiative vise à renforcer la vigilance face aux attaques par phishing, un vecteur majeur de compromission des données sensibles dans le secteur éducatif.

Vulnérabilité dans Docker Desktop (SSRF)

Résumé : Une vulnérabilité critique dans Docker Desktop permet à un attaquant d’exploiter une falsification de requêtes côté serveur (SSRF), exposant potentiellement les systèmes internes à des attaques. Cette faille nécessite une mise à jour urgente pour les organisations utilisant ce logiciel.

Vulnérabilités critiques dans les produits Microsoft

Résumé : Plusieurs vulnérabilités non spécifiées dans les produits Microsoft ont été découvertes, permettant à un attaquant de contourner les politiques de sécurité ou d’exécuter des actions malveillantes. Ces failles impactent des millions d’utilisateurs et nécessitent des correctifs immédiats.

Mardi 31 Mars 2026

Données post mortem : publication du cahier air2025 sur l’éthique des traces numériques

Résumé : La CNIL publie son cahier air2025, fruit du colloque d’octobre 2025 organisé avec la BnF, sur la gestion éthique des données post mortem. Ce document propose des recommandations pour encadrer l’accès et le traitement des données numériques après le décès d’un individu.

Exploitation active d’une faille critique dans Fortinet FortiClient EMS

Résumé : Une vulnérabilité de type injection SQL dans Fortinet FortiClient EMS est désormais exploitée activement par des attaquants. Cette faille permet une exécution de code à distance sans authentification, posant un risque majeur pour les infrastructures utilisant ce produit.

Attaque par supply chain sur un outil de développement (Axios)

Résumé : Une attaque par supply chain a ciblé un outil open-source (Axios) avec 100 millions de téléchargements hebdomadaires. Les chercheurs alertent sur le risque de compromission massive via des dépendances logicielles, soulignant l’importance de la vérification des chaînes d’approvisionnement.

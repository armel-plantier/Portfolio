Code de conduite RGPD pour le commerce de détail : validation par la CNIL

Résumé : La CNIL a approuvé un code de conduite porté par l’Alliance du Commerce, visant à aider les commerçants français (habillement, chaussures) à se conformer au RGPD. Ce cadre concret renforce la protection des données clients dans les processus de vente et de distribution, en traduisant les exigences réglementaires en actions opérationnelles.

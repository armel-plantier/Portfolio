### Court rules SAVE database illegal, orders it dismantled

Résumé : Un tribunal américain a jugé illégale la base de données SAVE (Systematic Alien Verification for Entitlements), la considérant comme une violation du Privacy Act, du Social Security Act et de l'Administrative Procedures Act. ### Decades-Old Squid Proxy Flaw ‘Squidbleed’ Can Expose User Data

Résumé : Une vulnérabilité critique baptisée "Squidbleed", similaire à Heartbleed, a été découverte dans le proxy Squid, utilisé par des millions de serveurs.

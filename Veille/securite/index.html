CVE-2026-23111 : Faille critique dans le noyau Linux permettant une élévation de privilèges locale

Résumé : Une faille critique (CVE-2026-23111, CVSS non attribué) dans le noyau Linux permet à un utilisateur local non privilégié d'obtenir un accès root via un problème de type *use-after-free* dans le code de filtrage de paquets nf_tables. Un exploit fonctionnel a été publié, exposant les systèmes Linux non patchés à des attaques locales. Le correctif a été intégré en amont le 5 février 2026.

CVE-2026-50751 : Exploitation active d'une faille critique dans Check Point VPN

Résumé : Une vulnérabilité critique (CVE-2026-50751, CVSS 9.3) dans les déploiements VPN et Mobile Access de Check Point, configurés avec le protocole obsolète IKEv1, est activement exploitée depuis mai 2026. L'attaque permet de contourner les mécanismes d'authentification et d'exécuter du code à distance. Un correctif est disponible via les mises à jour de sécurité Check Point.

Silent Ransom Group : Campagne d'extorsion ciblant les cabinets d'avocats américains

Résumé : Le groupe ransomware Silent Ransom Group combine *vishing*, usurpation d'identité IT et intrusions physiques pour voler des données et extorquer des cabinets d'avocats aux États-Unis. Les attaques exploitent des techniques de *fast flux DNS* pour masquer leur infrastructure de commande et contrôle. Aucune information sur une éventuelle fuite de données n'a été confirmée.

Shai-Hulud : Attaque en chaîne sur 19 packages PyPI

Résumé : Une campagne d'attaque *supply chain* (nom de code *Shai-Hulud*) a compromis 19 packages Python hébergés sur PyPI, cumulant des centaines de milliers de téléchargements. Les paquets malveillants exfiltraient des secrets de développeurs. Les packages infectés incluaient des noms liés à la science (ex: *scipy-utils*). Aucun correctif n'est disponible pour les systèmes déjà compromis.

NFCShare : Malware Android distribué via de fausses mises à jour bancaires sur GitHub

Résumé : Des variantes du malware Android *NFCShare* sont distribuées via de fausses mises à jour de applications bancaires légitimes hébergées sur GitHub. Le malware exploite des permissions abusives pour voler des données financières et exécuter des commandes à distance. Aucune application bancaire officielle n'est concernée, mais les utilisateurs doivent éviter les liens tiers.

Meta AI : Fuite de 20 000 comptes Instagram via un outil de support compromis

Résumé : Une faille dans l'outil d'IA de support de Meta a permis à des attaquants de réinitialiser les mots de passe de 20 225 comptes Instagram. L'attaque exploitait une défaillance dans la vérification par email lors des procédures de récupération de compte. Meta a depuis renforcé les contrôles d'authentification.

Vulnérabilité critique dans Cisco SD-WAN vManage (CVE-2026-20262)

Résumé : Cisco a corrigé une vulnérabilité critique (CVE-2026-20262, CVSS 9.1) dans Cisco SD-WAN vManage, exploitée en zero-day pour escalader des privilèges root. Les attaques ciblent les portails exposés, permettant une exécution de code arbitraire. Un correctif est disponible via les mises à jour logicielles.

Exploit critique dans Microsoft 365 Copilot (SearchLeak)

Résumé : Une chaîne d'exploits (SearchLeak) permet une fuite de données en un clic via Microsoft 365 Copilot Enterprise. Les attaquants peuvent voler emails, fichiers OneDrive/SharePoint et codes MFA via des URLs malveillantes. Microsoft a corrigé la faille, mais les attaques passées restent un risque.

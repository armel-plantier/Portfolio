Multiples vulnérabilités critiques dans Spring Framework (CVE-2026-38857, CVE-2026-38858)

Résumé : Plusieurs vulnérabilités critiques (CVE-2026-38857 et CVE-2026-38858) ont été découvertes dans Spring Framework, permettant à un attaquant distant de provoquer un déni de service (DoS) ou un contournement de la politique de sécurité via des requêtes malveillantes. Les systèmes exposés incluent les applications Java utilisant Spring Core, Spring Web et Spring Security. Un correctif est disponible dans les versions 6.1.12+ et 6.2.0+.

Les systèmes affectés incluent les versions de Xen antérieures à 4.18.1. Un patch est disponible via les mises à jour officielles.

Résumé : Une vulnérabilité critique (CVE-2026-5760, CVSS 9.8) dans SGLang permet une exécution de code à distance (RCE) via des fichiers GGUF malveillants.

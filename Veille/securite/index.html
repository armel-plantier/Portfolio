Vulnérabilité critique dans Google Chrome (CVE-2026-3909)

Résumé : Une vulnérabilité zero-day (CVE-2026-3909) dans Google Chrome est activement exploitée. Elle permet à un attaquant d'exécuter du code arbitraire à distance via des pages web malveillantes.

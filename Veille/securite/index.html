Multiples vulnérabilités critiques dans Microsoft Edge

Résumé : Plusieurs vulnérabilités critiques découvertes dans Microsoft Edge permettent une exécution de code arbitraire à distance (CVE-2026-3296, CVE-2026-3297, CVE-2026-3298), un contournement de la politique de sécurité (CVE-2026-3299) et des problèmes non spécifiés (CVE-2026-3300). Les systèmes affectés incluent toutes les versions de Microsoft Edge. Criticité : **Critique**. Un correctif est disponible via Windows Update.

Résumé : Un exploit fonctionnel (PoC) est désormais disponible pour la vulnérabilité DirtyDecrypt (CVE-2026-3301) dans le module rxgk du noyau Linux, permettant une élévation de privilèges SYSTEM.

Lundi 06 Juillet 2026

Multiples vulnérabilités critiques dans Roundcube (webmail)

Résumé : Plusieurs vulnérabilités critiques ont été découvertes dans Roundcube, un client webmail open-source. Ces failles permettent à un attaquant distant d'exécuter un déni de service (DoS), une falsification de requêtes côté serveur (SSRF) et une injection de code indirecte (XSS). Les systèmes exposés incluent les serveurs Roundcube non patchés. Un correctif est disponible via la mise à jour vers la version 1.6.8 ou ultérieure.

Exploitation en cours d'une faille critique dans Adobe ColdFusion (CVE-2026-48282)

Résumé : Une vulnérabilité de criticité maximale (CVE-2026-48282) dans Adobe ColdFusion est désormais exploitée activement par des attaquants. Cette faille permet l'exécution de code à distance (RCE) sur les serveurs vulnérables. Les versions affectées incluent ColdFusion 2021 et 2023. Adobe a publié un correctif urgent (APSB26-45).

Nouvelle faille critique dans Linux KVM (CVE-2026-53359) permettant l'échappement de VM

Résumé : Une vulnérabilité critique (CVE-2026-53359, "Januscape") dans le module KVM de Linux permet à une machine virtuelle invitée d'échapper à son isolation et de corrompre l'état du noyau hôte. La faille affecte les systèmes Intel et AMD x86. Un correctif est disponible dans le noyau Linux 6.9.7+.

Fuite de données via émissions radio depuis des systèmes air-gap (TrojPix)

Résumé : Des chercheurs ont démontré TrojPix, une attaque permettant d'exfiltrer des données depuis des systèmes isolés (air-gap) via des émissions radio générées par des câbles vidéo modifiés. La technique exploite des variations imperceptibles de pixels. Aucune contre-mesure matérielle n'est actuellement déployée.

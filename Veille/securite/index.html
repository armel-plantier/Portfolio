Ghost CMS CVE-2026-26980 exploité pour pirater plus de 700 sites

Résumé : Une faille critique d'injection SQL (CVE-2026-26980, CVSS 9.4) dans Ghost CMS est activement exploitée pour injecter du JavaScript malveillant et lancer des attaques ClickFix. Des sites majeurs comme ceux d'Harvard, Oxford et DuckDuckGo ont été compromis. Aucun correctif officiel n'est encore disponible à ce stade.

Kali365 : Une plateforme de phishing-as-a-service cible Microsoft 365

Résumé : Le FBI alerte sur Kali365, une plateforme Phishing-as-a-Service (PhaaS) exploitant l'authentification OAuth via code appareil pour voler des tokens de session et contourner la MFA sur Microsoft 365. Cette méthode réduit les barrières pour les cybercriminels. Aucun patch n'est mentionné pour cette technique spécifique.

Résumé : Une campagne coordonnée (TrapDoor) a distribué des packages malveillants sur npm, PyPI et Crates.io pour voler des identifiants. Plus de 34 packages infectés couvrant 384 versions ont été identifiés. Les victimes doivent vérifier leurs dépendances et appliquer les correctifs disponibles.

Mégaodon : Plus de 5 500 dépôts GitHub infectés par une attaque par chaîne d'approvisionnement

Résumé : Une attaque par chaîne d'approvisionnement (Megalodon) a infecté plus de 5 500 dépôts GitHub via des commits automatisés malveillants. Ces commits injectaient des workflows GitHub Actions pour voler des identifiants, clés et tokens CI. Aucune solution globale n'est disponible, mais une revue des workflows est urgente.

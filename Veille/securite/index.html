TrapDoor : Attaque par chaîne d'approvisionnement logiciel touchant npm, PyPI et Crates.io

Résumé : Une campagne coordonnée nommée TrapDoor a ciblé les écosystèmes npm, PyPI et Crates.io pour distribuer des malwares volant des identifiants. Plus de 34 paquets malveillants répartis sur 384 versions ont été identifiés, avec une activité initiale enregistrée dès le 22 mai 2026. Les attaquants exploitent des packages apparemment légitimes pour infiltrer les environnements de développement.

CVE-2026-26980 : Exploitation massive de Ghost CMS pour des attaques ClickFix

Résumé : Une vulnérabilité critique d'injection SQL (CVE-2026-26980, CVSS 9.4) dans Ghost CMS est exploitée pour injecter du code JavaScript malveillant, compromettant plus de 700 sites web, dont ceux de Harvard, Oxford et DuckDuckGo. Les attaquants utilisent cette faille pour rediriger les visiteurs vers des campagnes ClickFix et voler des données sensibles.

FBI : Plateforme Kali365 exploitant OAuth pour pirater Microsoft 365

Résumé : Le FBI alerte sur Kali365, une plateforme Phishing-as-a-Service (PhaaS) abusant de l'authentification OAuth via des codes de périphérique pour voler des tokens de session et contourner la MFA sur Microsoft 365. Cette méthode réduit la complexité des attaques et cible spécifiquement les comptes professionnels.

Résumé : Le groupe nord-coréen Lazarus utilise RemotePE, un malware multiplateforme (Windows/Linux/macOS) pour infiltrer les entreprises financières et les plateformes cryptographiques. CERT-In impose un délai de 12 heures pour corriger les failles exposées aux attaques IA

Résumé : L'autorité indienne CERT-In recommande aux organisations de corriger les vulnérabilités critiques des systèmes exposés à Internet dans un délai de 12 heures après leur signalement, en raison de l'automatisation croissante des attaques par des outils d'IA. Cette mesure vise à limiter l'exploitation massive des failles non patchées.

Résumé : Microsoft a publié un correctif pour une vulnérabilité de type RCE (CVE-2026-45659) affectant toutes les versions de SharePoint. KnowledgeDeliver : Zero-day exploité pour déployer Godzilla et Cobalt Strike

Résumé : Une faille zero-day (ViewState deserialization) dans KnowledgeDeliver, un LMS populaire au Japon, a été exploitée pour déployer le web shell Godzilla et Cobalt Strike. La vulnérabilité, liée à des valeurs machineKey codées en dur, permet une exécution de code à distance et une prise de contrôle complète des serveurs compromis.

7-Eleven : Fuite de données de 185 000 personnes attribuée à ShinyHunters

Résumé : Le gang ShinyHunters revendique le vol des données personnelles de 185 000 clients de 7-Eleven, incluant noms, adresses, emails et dates de naissance.

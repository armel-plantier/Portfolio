Vulnérabilité critique dans Citrix NetScaler (CVE-2026-3055)

Patch : Disponible via les mises à jour officielles Citrix.

DeepLoad : Nouveau malware IA volant des identifiants et contournant la détection

Résumé : Le malware DeepLoad utilise des techniques d'obfuscation basées sur l'IA (génération de code aléatoire) et la persistance WMI pour voler les identifiants du navigateur. Il combine la tactique ClickFix avec des injections de processus pour éviter la détection statique. Systèmes concernés : Windows. Criticité : Élevée. Patch : Non applicable (malware).

Copy Fail : cette redoutable faille Linux permet d’obtenir un accès root

Résumé : Une faille critique nommée Copy Fail (CVE-2026-31431) a été découverte dans le noyau Linux, offrant un accès root non autorisé sur les distributions depuis 2017. Les administrateurs doivent appliquer les correctifs urgents pour éviter une compromission des serveurs. La vulnérabilité affecte les versions 5.x à 6.x du noyau.

Critical cPanel Authentication Vulnerability Identified — Update Your Server Immediately

Résumé : Une faille d'authentification critique (non encore nommée CVE) a été identifiée dans toutes les versions supportées de cPanel. Elle permet à un attaquant d'obtenir un accès non autorisé à l'interface WHM. Les administrateurs doivent appliquer les mises à jour disponibles immédiatement pour éviter une compromission des serveurs de production. La vulnérabilité est classée comme "critique" par les éditeurs.

Lundi 28 Avril 2026

Cette faille GitHub est exploitable par un simple Git Push (CVE-2026-3854)

Résumé : Une vulnérabilité critique (CVE-2026-3854, CVSS 8.7) dans GitHub.com et GitHub Enterprise Server permet une exécution de code à distance (RCE) via une simple commande git push. Les administrateurs DevOps doivent appliquer les correctifs urgents et vérifier les permissions des dépôts pour éviter une compromission des pipelines CI/CD. La faille est déjà exploitée en masse par des groupes cybercriminels.

Azure Linux 4.0 : Microsoft officialise sa première distribution Linux basée sur Fedora

Résumé : Microsoft a dévoilé Azure Linux 4.0, une distribution Linux open source et gratuite basée sur Fedora, conçue pour les environnements cloud et hybrides. Elle est compatible avec WSL sur Windows 11, offrant une intégration native pour les développeurs et administrateurs système travaillant sur des infrastructures Microsoft. Cette solution simplifie la gestion des conteneurs et des workloads Linux dans les environnements Azure.

npm Adds 2FA-Gated Publishing and Package Install Controls Against Supply Chain Attacks

Résumé : GitHub a introduit des contrôles de sécurité renforcés pour npm, incluant la publication conditionnelle par 2FA et une validation explicite des paquets avant leur mise à disposition publique. Packagist Supply Chain Attack Infects 8 Packages Using GitHub-Hosted Linux Malware

Résumé : Une attaque coordonnée sur Packagist a infecté 8 paquets Composer avec du code malveillant récupérant un binaire Linux depuis GitHub. Bien que les paquets affectés ne modifiaient pas directement les fichiers `composer.json`, ils exposaient les systèmes à des risques d'exécution de code arbitraire. Les administrateurs doivent vérifier leurs dépendances et utiliser des outils comme `dependabot` pour détecter les anomalies.

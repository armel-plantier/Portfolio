### Docker CVE-2026-34040 : faille critique permettant un accès root sur l'hôte

Résumé : Une faille de sécurité critique (CVE-2026-34040, CVSS 8.8) dans Docker Engine permet à un attaquant de contourner les plugins d'autorisation (AuthZ) et d'obtenir un accès root sur l'hôte. Cette vulnérabilité exploite une correction incomplète d'une faille précédente (CVE-2024-41110). Les administrateurs doivent appliquer les correctifs immédiatement pour éviter des compromissions de conteneurs et d'infrastructures.

### Python : compromission de la supply chain via le package litellm

Résumé : Une compromission de la supply chain a été détectée dans le package Python `litellm` (version 1.82.8) sur PyPI. La roue malveillante contient un fichier `.pth` corrompu qui exécute du code arbitraire lors de l'installation. Les équipes DevOps doivent auditer leurs dépendances et bloquer les versions suspectes via des outils comme `pip-audit` ou des politiques de sécurité renforcées.

### Google : transition vers la cryptographie post-quantique d'ici 2029

### APT28 exploite des routeurs SOHO pour des attaques de hijacking DNS

Résumé : Le groupe russe APT28 (Forest Blizzard) a compromis des routeurs MikroTik et TP-Link pour les transformer en infrastructure malveillante dans une campagne d'espionnage mondial. Les routeurs infectés modifient les paramètres DNS pour rediriger le trafic vers des serveurs contrôlés par les attaquants. Les administrateurs réseau doivent auditer les configurations DNS, désactiver les services inutiles (comme UPnP) et appliquer les correctifs firmware.

### GPUBreach : nouvelle attaque RowHammer contre les GPU

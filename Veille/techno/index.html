Trois failles critiques dans UniFi OS Server permettent une élévation de privilèges root

Résumé : Trois vulnérabilités (CVE-2026-34908, CVE-2026-34909, CVE-2026-34910) permettent d'exécuter du code arbitraire en tant que root sur UniFi OS Server. Résumé : Un bug de type "use-after-free" dans le code nf_tables du noyau Linux (CVE-2026-23111) permet à un utilisateur local non privilégié d'obtenir des privilèges root et de s'échapper de conteneurs. Un exploit fonctionnel est désormais public, nécessitant une mise à jour immédiate des systèmes concernés.

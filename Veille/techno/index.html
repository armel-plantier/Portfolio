Noyau Linux 7.2 : optimisations majeures pour les CPU modernes et amélioration des performances

Résumé : Linux 7.2 introduit le support du Cache Aware Scheduling pour les CPU Intel/AMD modernes, réduisant les latences et boostant les performances. La lecture de /proc/filesystems est accélérée jusqu'à 444%, et le support de WebAssembly est validé par le GCC Steering Committee. Une mise à jour critique pour les administrateurs gérant des infrastructures haute performance.

LiteLLM : vulnérabilités critiques permettant une prise de contrôle de serveurs IA

Résumé : Une chaîne de 3 vulnérabilités dans LiteLLM (proxy IA open-source) permet à un utilisateur à faible privilège de devenir administrateur et d'exécuter du code. LiteLLM gère plus de 100 fournisseurs de modèles IA, rendant cette faille critique pour les environnements DevOps utilisant des passerelles IA.

GCC : support officiel de WebAssembly pour compiler C/C++ en WASM

Résumé : Le GCC Steering Committee valide l'ajout d'un backend WebAssembly, permettant de compiler du code C/C++ directement en WASM. Cette fonctionnalité simplifie le développement d'applications multiplateformes et le déploiement de services légers. Un atout pour les équipes DevOps travaillant avec des conteneurs WASM.

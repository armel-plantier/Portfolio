### Docker CVE-2026-34040 Lets Attackers Bypass Authorization and Gain Host Access

Résumé : Une vulnérabilité critique (CVE-2026-34040, CVSS 8.8) dans Docker Engine permet de contourner les plugins d'autorisation (AuthZ) via des requêtes HTTP spécifiques, offrant un accès root à l'hôte. Cette faille exploite une correction incomplète pour CVE-2024-41110 et nécessite une attention immédiate pour les environnements de production.

Cette vulnérabilité matérielle nécessite une mitigation urgente via des correctifs firmware ou des protections logicielles.

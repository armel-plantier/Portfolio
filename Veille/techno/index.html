Januscape : 16 ans de faille critique dans KVM

Résumé : Januscape (CVE-2026-53359), une faille use-after-free découverte après 16 ans dans KVM, permet à une VM Intel ou AMD de planter son hôte ou de s'en échapper. Cette vulnérabilité critique dans le code du shadow MMU affecte les systèmes virtualisés et nécessite une mise à jour immédiate du noyau Linux.

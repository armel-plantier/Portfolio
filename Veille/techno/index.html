LiteLLM : Chaîne de vulnérabilités permettant une prise de contrôle de serveurs de passerelle IA

Résumé : Une chaîne de 3 vulnérabilités dans LiteLLM (proxy open source pour l'IA) permet à un utilisateur à faible privilège de devenir administrateur et d'exécuter du code sur le serveur.

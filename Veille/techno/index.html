F5 corrige deux vulnérabilités critiques dans NGINX Open Source

Résumé : F5 a publié des correctifs pour deux vulnérabilités critiques (CVE-2026-42530 et CVE-2026-42531) dans NGINX Open Source, permettant une exécution de code à distance. La première est une faille de type *use-after-free* dans le module HTTP/3, tandis que la seconde concerne une corruption de mémoire dans le traitement des requêtes. Les administrateurs doivent appliquer les mises à jour immédiatement pour éviter des attaques par déni de service ou exécution de code arbitraire.

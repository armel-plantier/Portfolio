GhostLock : faille Linux critique depuis 15 ans

Résumé : Une faille critique (CVE-2026-43499) dans le noyau Linux, découverte via IA, permet à un utilisateur local d'obtenir les privilèges root sur la majorité des distributions. Présente depuis 2011, cette vulnérabilité affecte les systèmes non patchés et nécessite une mise à jour immédiate du noyau.

HTTP QUERY : nouvelle méthode HTTP depuis 2010

Résumé : Le RFC 10008 officialise la méthode HTTP QUERY, combinant les avantages de GET (safe) et POST (corps de requête). Cette évolution simplifie les interactions API et réduit les risques liés aux requêtes POST traditionnelles. Les administrateurs doivent adapter leurs proxys et règles de filtrage.

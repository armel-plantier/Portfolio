Faille critique dans NGINX : CVE-2026-42533 menace la stabilité des serveurs web

Résumé : F5 a corrigé la CVE-2026-42533, une faille critique dans NGINX qui peut faire planter les workers et, dans certains cas, permettre une exécution de code à distance. Cette vulnérabilité affecte les configurations utilisant des expressions régulières dans les directives `map` ou `server`. Une mise à jour immédiate est fortement recommandée pour les administrateurs exploitant NGINX en production.

HollowByte : faille DoS silencieuse dans OpenSSL (11 octets)

Résumé : La faille HollowByte, corrigée depuis OpenSSL 4.0.1, permet de saturer la mémoire d'un serveur avec un paquet de seulement 11 octets.

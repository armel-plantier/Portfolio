NixOS et Sécurix : une alternative Linux souveraine pour l'État français

Résumé : L'État français envisage d'adopter NixOS via le projet Sécurix pour remplacer Windows sur les postes des agents publics. Cette distribution immutable offre une gestion déclarative des configurations, idéale pour la reproductibilité et la sécurité des environnements critiques. Son modèle de configuration modulaire (Nixpkgs) simplifie aussi la maintenance à grande échelle.

Amazon S3 Files : le stockage objet S3 se transforme en système de fichiers natif

Résumé : AWS annonce Amazon S3 Files, une fonctionnalité permettant d'utiliser des buckets S3 comme des systèmes de fichiers classiques (via EFS). Cette évolution simplifie l'intégration des données cloud dans les workflows locaux et réduit la complexité des migrations hybrides, tout en conservant les avantages du stockage objet (scalabilité, durabilité).

GreenBoost-Proton : étendre la VRAM des GPU NVIDIA avec la RAM système

Résumé : GreenBoost, solution open-source de tiering mémoire, propose désormais GreenBoost-Proton pour les jeux sous Linux. Cette technologie permet d'utiliser la RAM système et le stockage NVMe comme extension de la VRAM des GPU NVIDIA, idéale pour les configurations limitées en mémoire graphique. Une avancée majeure pour les environnements HPC et gaming.

Un pas important pour les administrateurs gérant des parcs de machines.

FTRFS : un nouveau système de fichiers tolérant aux fautes pour Linux

Résumé : FTRFS (Fault-Tolerant File-System) est proposé comme une alternative aux systèmes existants (Btrfs, ZFS) avec une approche axée sur la résilience aux pannes matérielles. Son architecture vise à réduire les risques de corruption de données en cas de défaillances critiques, un atout pour les environnements critiques (bases de données, stockage distribué).

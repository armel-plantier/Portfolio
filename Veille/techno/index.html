Docker CVE-2026-34040 : Faille critique permettant un accès root sur l'hôte

Résumé : Une faille de sécurité critique (CVE-2026-34040, score CVSS 8.8) dans Docker Engine permet de contourner les plugins d'autorisation (AuthZ) et d'obtenir un accès root sur l'hôte. Cette vulnérabilité, issue d'un correctif incomplet pour CVE-2024-41110, expose les environnements conteneurisés à des risques d'escalade de privilèges. Les administrateurs doivent appliquer les correctifs urgents et auditer leurs configurations AuthZ.

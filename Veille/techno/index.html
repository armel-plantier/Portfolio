Microsoft annonce la transition d'Entra Connect Sync vers Cloud Sync

Résumé : Microsoft accélère la migration des environnements hybrides vers le cloud avec la fin d'Entra Connect Sync au profit d'Entra Cloud Sync. Cette évolution simplifie la synchronisation des identités entre Active Directory et Entra ID, mais impose une refonte des processus existants. Les administrateurs doivent anticiper la migration pour éviter des interruptions de service.

Démantèlement du botnet Glassworm ciblant GitHub et VS Code

Résumé : Une collaboration entre CrowdStrike, Google et la Shadowserver Foundation a permis de neutraliser Glassworm, un botnet exploitant des packages malveillants sur GitHub et VS Code. Cette opération sécurise la supply chain des développeurs en éliminant une menace persistante de vol de données et d'exécution de code à distance. Les administrateurs doivent vérifier l'intégrité de leurs environnements CI/CD.

CVE-2026-0257 : Exploitation active d'une faille d'authentification sur PAN-OS

Résumé : Palo Alto Networks alerte sur une faille critique (CVE-2026-0257, CVSS 7.8) dans PAN-OS et Prisma Access, permettant un contournement d'authentification sur GlobalProtect. IBM et Red Hat investissent 5 milliards $ dans Project Lightwell pour sécuriser la supply chain logicielle

Résumé : Le projet Lightwell vise à renforcer la sécurité des chaînes d'approvisionnement logicielles via une collaboration avec 20 000 ingénieurs. L'initiative inclut des outils de détection des vulnérabilités dans les dépendances, des mécanismes de signature cryptographique renforcés et une intégration native avec les plateformes DevOps. Pertinent pour les équipes DevSecOps cherchant à réduire les risques de dépendances malveillantes.

Vulnérabilité ChatGPhish exploitant les résumés web de ChatGPT

Résumé : Une faille (ChatGPhish) dans ChatGPT permet aux attaquants d'injecter des liens Markdown malveillants dans les résumés web générés par l'IA.

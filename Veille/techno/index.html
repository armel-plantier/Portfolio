Dimanche 18 Mai 2026

Microsoft Cloud-Initiated Driver Recovery pour Windows

Résumé : Microsoft introduit "Cloud-Initiated Driver Recovery" dans Windows Update pour automatiser la réparation des pilotes défectueux, réduisant ainsi les interventions manuelles des administrateurs système. Cette solution cloud-based permet de détecter et corriger les incompatibilités sans redémarrage forcé.

MiniPlasma : Zero-Day Windows avec privilèges SYSTEM

Résumé : La faille zero-day "MiniPlasma" permet une élévation de privilèges SYSTEM sur des systèmes Windows 11 à jour, exploitable via des outils comme Chaotic Eclipse. Les administrateurs doivent appliquer les correctifs urgents ou désactiver les services vulnérables en attendant.

YellowKey : Exploit BitLocker via accès physique

Résumé : La faille "YellowKey" contourne les protections BitLocker par accès physique au système, nécessitant une intervention manuelle pour être exploitée. Les administrateurs doivent renforcer la sécurité physique des postes et surveiller les tentatives d’accès non autorisées.

Pwn2Own Berlin 2026 : 47 failles zero-day découvertes

Résumé : L’édition 2026 du Pwn2Own a révélé 47 vulnérabilités zero-day dans VMware, Windows, des outils IA et d’autres technologies critiques. Les administrateurs doivent prioriser les mises à jour des solutions concernées pour éviter les exploits en production.

Résumé : Linux 7.2 introduit le flag OPENAT2_REGULAR pour sécuriser l’appel système openat2 contre les attaques par tromperie de programmes sécurisés. Les administrateurs doivent tester cette mise à jour pour éviter les régressions dans les environnements critiques.

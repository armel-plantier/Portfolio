Coreutils pour Windows : exécution native des commandes Linux

Résumé : Microsoft annonce Coreutils pour Windows, un projet open source permettant d'exécuter nativement des commandes Linux (grep, sed, awk, etc.) sur Windows. Cette intégration facilite la migration des scripts et workflows DevOps entre systèmes, réduisant la dépendance aux environnements WSL ou Cygwin.

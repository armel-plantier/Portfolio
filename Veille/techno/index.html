1Password for Claude : Connexion sécurisée sans partage de mots de passe

Résumé : 1Password intègre une fonctionnalité permettant à l'IA Claude de se connecter à vos comptes via un navigateur sans jamais exposer vos identifiants ou codes MFA. 7-Zip 26.02 : Correction d'une RCE via des archives XZ malveillantes

Résumé : La version 26.02 de 7-Zip corrige CVE-2026-14266, une vulnérabilité de débordement de tampon dans le décodage XZ. FakeGit : Campagne massive de 7 600 dépôts GitHub malveillants distribuant SmartLoader

Résumé : Une campagne d'attaque (FakeGit) a infiltré 7 600 dépôts GitHub, dont 800 se faisant passer pour des outils IA ou des serveurs MCP. Ces dépôts distribuent le malware SmartLoader, capable de compromettre les machines des développeurs. Une vigilance accrue est nécessaire lors de l'import de dépendances externes.

AVX-512 fait son retour avec les futurs CPU Intel Nova Lake

Résumé : Intel réintroduit le support AVX-512 dans ses futurs CPU Nova Lake, incluant les cœurs P et E. Intel dévoile une architecture mémoire XBM pour remplacer l'HBM

Résumé : Intel brevette une architecture mémoire XBM (eXtended Backend Memory) qui élimine le besoin d'interposeurs silicium coûteux comme dans l'HBM. Utilisant des liens UCIe et une réparation intégrée, cette solution réduit les coûts de packaging et pourrait atténuer les goulots d'étranglement mémoire dans les serveurs IA. Une alternative disruptive aux solutions actuelles.

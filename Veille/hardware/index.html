Nvidia lance le superchip RTX Spark pour Windows sur Arm

Résumé : Nvidia dévoile le RTX Spark, un superchip combinant un CPU Arm 20 cœurs Grace et un GPU Blackwell, avec jusqu'à 128 Go de mémoire unifiée. Cette plateforme vise à transformer Windows en un OS agentique pour l'IA, avec support natif des technologies anti-triche (Fortnite, Valorant) et DRM. Microsoft collabore pour une intégration optimale. Impact : menace directe pour les architectures x86 dans les postes haut de gamme et serveurs légers.

Impact : renforcement de la position d'Intel dans les datacenters pour le calcul intensif et l'IA.
